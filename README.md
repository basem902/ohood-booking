# حجوزات عهود 🗓️

نظام إدارة مواعيد وحجوزات — تطبيق ويب **(PWA)** يعمل بدون إنترنت، وبياناتك محفوظة على جهازك عبر قاعدة بيانات **IndexedDB** (لا تُرفع لأي خادم).

## 🔗 الرابط المباشر
**https://basem902.github.io/ohood-booking/**

## ✨ المزايا
- تسجيل: اسم العميل، الجوال، التاريخ، الوقت، **المدة**، الباقة، السعر، **المدفوع**، الحالة، والملاحظات.
- **تحذير التعارض** في نفس اليوم (حسب المدة) مع خيار **منع الحجز المزدوج**.
- **تقويم شهري/أسبوعي** يُبرز الأيام المزدحمة والمتعارضة.
- **تقارير شهرية** + حالة الدفع والمبلغ المتبقّي.
- **إشعارات تذكير** بالمتصفح + زر تذكير **واتساب**.
- قابل **للتثبيت كتطبيق** (PWA) ويعمل أوفلاين.
- **نسخ احتياطي** (تصدير/استيراد JSON) وتصدير **Excel** (CSV).

## 🚀 التشغيل
- **عبر الإنترنت:** افتح [الرابط المباشر](https://basem902.github.io/ohood-booking/).
- **محليًا على ويندوز:** شغّل `تشغيل-حجوزات-عهود.bat` (يتطلب Node.js) ليفتح على `http://localhost:8080`.
- التطبيق مجرد ملفات ثابتة، فيمكن استضافته على أي خادم ثابت.

## 🛠️ التقنية
HTML / CSS / JavaScript صرف — بدون أي مكتبات خارجية. قاعدة بيانات IndexedDB، و Service Worker + Web App Manifest لدعم PWA والعمل دون اتصال.
